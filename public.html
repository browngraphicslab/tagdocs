Welcome!

Welcome to the Touch Art Gallery User Reference. This site contains a collection of resources for setting up TAG, taking advantage of its capabilities, and installing a TAG server. There is also a list of features under active development at the bottom. This may be useful if you are contemplating using Touch Art Gallery. If you have any ideas for new features, or if you have noticed any erratic functionality, please email us at brown.touchartgallery@outlook.com; we would be more than happy to talk with you!

Windows 8/8.1 Application Setup
The TAG Windows app is available in the Windows App Store here. If you are using a Windows 8/8.1 machine, you should be able to install directly from there.

By default, TAG connects to the Brown Graphics Lab demo server, which has a selection of demo content to explore. To explore content on another server, you can use the "Change Server" button in the bottom right of the start page. You will need to enter the address of this server (a couple sample addresses are provided above).
